Administrator Guide

Administrators manage cipher records and user access.
  • Add Cipher Records
  • Edit Cipher Records
  • Delete Cipher Records
  • Authorize Users
  • Manage Access Control
  • Search Records

  1. Open Admin Dashboard.
  2. Click Authorise User.
  3. Enter Employee ID OR Personal No.
  4. Enter Designation.
  5. Enter Section.
  6. Save the User.
Only authorized users can register in the application.

  1. Click Add Record.
  2. Enter Name.
  3. Enter Agent Name.
  4. Enter IP Address
  5. Enter Section.
  6. Upload Cipher Key (.txt file).
  7. Submit the form.
Validation Rules
  • Agent Name must be unique.
  • Cipher Key must be unique.
  • Only .txt files are accepted.

Administrators can update:

  • Name
  • IP Address
  • Section
  • Agent Name
Agent Name and Cipher Key are protected fields and cannot be modified.

  1. Click Delete.
  2. Confirm in the popup window.
  3. Record will be removed permanently.
Deleted records cannot be recovered.

Employee ID must contain exactly 6 characters.

Examples:
  • 815027
  • maint1
  • maint2
  • 100059

User Guide

  1. Request authorization from Administrator.
  2. Administrator adds your Employee ID.
  3. Open Registration page.
  4. Enter Employee ID.
  5. Create Username and Password.
  6. Login to the application.

Use the search box to find records by:
  • Name
  • Agent Name
  • IP Address
  • section

  1. Click the Eye icon.
  2. Popup window will open.
  3. View the complete Cipher Key.
  4. Use Copy button if needed.

Cipher Keys are masked by default. 
asd******bnv
Only the first 3 and last 3 characters are displayed.

Do not share Cipher Keys with unauthorized personnel.
